Etiquetas HTML

Este ejemplo muestra cómo combinar algunas de las etiquetas más básicas de HTML5.
Recuerda que es importante entender la diferencia entre ellas

Etiqueta ul+li

Si listamos personas nominadas a los Oscars, dado que el orden no altera el significado.
debemos usar ul.

  • David Verdaguez
  • Jesús Castro
  • Israel Elejalde
  • Dani Rivera

Etiqueta ol+li

En el caso de que estemos listando elementos donde el orden es importante, como por
ejemplo la clasificación de un mundial del futbol, debemos usar ol.

    1.España
    2.Paises Bajos
    3.Alemania
    4.Uruguay
